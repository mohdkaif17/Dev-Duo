# Dev-Duo
🚀 TechVerse – Club Hub Platform
🌐 Overview

TechVerse is a full-featured, modern Club Hub web application designed to serve as the central digital platform for college technical communities.

It brings together events, challenges, projects, team management, and analytics into one unified ecosystem, eliminating scattered communication across multiple platforms.

🎯 Problem Statement

College technical clubs often lack a centralized system to manage their activities. Information is scattered across WhatsApp, Google Forms, and social media, leading to:

❌ Poor event visibility
❌ No structured challenge system
❌ Lack of project showcasing
❌ No performance tracking
❌ Limited team visibility

👉 TechVerse solves this by providing a single, interactive platform for all club activities.

💡 Solution

TechVerse is a one-stop digital hub that enables:

📅 Event discovery & registration
🧩 Challenge participation & submissions
💼 Project showcasing
👥 Team visibility
📊 Leaderboards & analytics
